将hexo部署到github上

花了两个小时，才搭建成功，作为一名自称混迹IT圈的人来说，真是够了。好吧，记录一下搭建过程，以便让大家更快上手。

1.前期准备，包括git，nodejs，github账号，ssh设置等，自行百度，谷歌。

2.安装hexo，这个步骤有条件的同学最好翻墙，我装的时候各种下不下来，一翻就ok。

3.正式步骤开始咯，首先，新建一个文件夹，我取名为blog

4.进入blog，右键，gitbash，分别执行命令hexo init ，npm install,hexo g

5.修改_config.yml文件（hexo3.0的和以前的版本有所不同，注意type为git）

6.npm install hexo-deployer-git --save

7.hexo d 部署到github上。
